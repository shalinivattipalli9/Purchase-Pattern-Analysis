# Purchase Pattern Analysis (Market Basket Analysis)

## 📌 Project Overview
This project performs Market Basket Analysis on retail transaction data using the Apriori algorithm. The goal is to identify frequently purchased product combinations and generate actionable business insights.

## 🔍 Key Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Items per Transaction Analysis
- Top Selling Products Identification
- Frequent Itemset Generation
- Association Rule Mining
- Lift and Confidence Analysis
- Business Recommendations

## 🛠 Tools & Technologies Used
- Python
- Pandas
- MySQL
- MLxtend (Apriori Algorithm)
- Matplotlib
- Seaborn

## 📊 Key Insights
- Most transactions contain a small number of items.
- Strong association rules identified with high lift values.
- High confidence rules suggest bundling opportunities.
- Cross-selling strategies recommended for high-lift product pairs.

## 🚀 Outcome
The project helps businesses improve sales strategies, optimize product placement, and enhance customer satisfaction through data-driven insights.
